## Installation
To use this dotfiles repository simply clone it with:  
`git clone git://github.com/butkej/dotfiles.git`

Afterwards, simply change into the dotfiles directory, make the script executable and run it:  
```
cd ~/dotfiles
chmod +x install.sh
./install.sh
```
